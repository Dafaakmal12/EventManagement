# how to run this project

## Stack
- NPM (Node Package Manager)
- NodeJS
- React
- Vite
- TailwindCSS

## How to run
- Clone this repository
- Open terminal and go to this project directory
- Run `npm install`
- Run `npm run dev`
- Open your browser and go to `localhost:3000`

## How to build
- Run `npm run build`
- Run `npm run serve`
- Open your browser and go to `localhost:5000`
